Rock Paper Scissors Game
A simple and interactive Rock Paper Scissors game built with React. Play against the computer and track your scores.

Features
•	Play Rock, Paper, or Scissors against the computer
•	Real-time score tracking (wins, losses, ties)
•	Clean and responsive design
•	Reset score functionality

How to Play
1.	Click on Rock, Paper, or Scissors to make your move
2.	The computer will randomly select its move
3.	The game will determine the winner based on classic rules:
     -Rock beats Scissors
     -Scissors beats Paper
     -Paper beats Rock
     -Same moves result in a tie
4.	Your score will update automatically
5.	Use "Reset Score" to start over
   
Installation and Setup
1.	Make sure you have Node.js installed on your computer
2.	Navigate to the project directory: cd rock-paper-scissors
3.	Install the required dependencies: npm install
4.	Start the development server: npm start
5.	Open your web browser and go to:http://localhost:3000
   
Project Structure
The main files are:
•	src/App.js - Contains the game logic and components
•	src/App.css - Contains the styling for the game
•	public/index.html - The main HTML file

Technologies Used
•	React
•	JavaScript
•	CSS
•	HTML

Available Commands
•	npm start - Runs the app in development mode
•	npm test - Runs the test suite
•	npm run build - Builds the app for production

How to Customize	
You can easily modify the game by editing the App.js file for game logic or App.css for styling. The code is straightforward and well-organized.

Browser Support	
Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
